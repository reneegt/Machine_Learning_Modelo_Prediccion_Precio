# Machine_Learning_Modelo_Prediccion_Precio
Repositorio del proyecto Rusty Bargain Desarrollo de un modelo de Machine Learning para una app de venta de coches de segunda mano. El objetivo es predecir rápidamente el valor de mercado de un vehículo basándose en su historial, especificaciones técnicas y equipamiento, optimizando el tiempo de predicción y la precisión (REQM)
